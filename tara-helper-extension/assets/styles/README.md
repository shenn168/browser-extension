# 🛡️ TARA Helper - ISO/SAE 21434 Threat Analysis Assistant

**Version:** 1.0.0  
**Last Updated:** January 15, 2024  
**Compatible with:** Microsoft Edge (Chromium-based)

## Overview

TARA Helper is an intelligent browser extension that accelerates automotive cybersecurity threat analysis and risk assessment (TARA) by automatically extracting, structuring, and pre-populating ISO/SAE 21434-compliant threat scenarios from security reports, advisories, and technical documentation in real-time.

## 🌟 Key Features

- **Intelligent Content Analysis**: Automatically identifies assets, threats, attack paths, and controls from web pages
- **ISO/SAE 21434 Compliance**: Generates threat scenarios aligned with ISO/SAE 21434 standards
- **Attack Feasibility Calculator**: Calculates attack feasibility based on ISO 21434 criteria
- **Interactive Highlighting**: Color-coded page highlighting for identified elements
- **Comprehensive Sidebar**: Full analysis panel with tabbed interface
- **Export Capabilities**: Export analysis as JSON or CSV
- **Risk Assessment**: Automated risk level determination
- **Control Recommendations**: Suggests appropriate cybersecurity controls

## 📋 Prerequisites

- **Microsoft Edge** (version 88 or later) - Chromium-based
- **Operating System**: Windows 10/11, macOS, or Linux
- **Permissions**: Active tab access, storage, side panel
