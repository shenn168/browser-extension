# 🪷 Zen Quotes Browser Extension

A beautiful Microsoft Edge browser extension that delivers random inspirational zen quotes with a single click. Find your moment of peace and wisdom throughout your busy day.

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Browser](https://img.shields.io/badge/browser-Microsoft%20Edge-0078D7.svg)

## ✨ Features

- **Random Inspirational Quotes**: Fetches fresh zen quotes from the ZenQuotes API
- **Beautiful UI**: Modern, calming dark theme with smooth animations
- **Offline Support**: Built-in fallback quotes when API is unavailable
- **Lightweight**: Minimal permissions required
- **Fast**: Instant quote display with elegant loading states
